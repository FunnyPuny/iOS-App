### FunnyPuny

FunnyPuny is an easy-to-use app that helps you to create or maintain your everyday routine and achieve your long-term goals!

### Tech stacks

Core:
-   SnapKit
-   Realm

Others:
-   Swift Package Manager
-   SwiftLint
-   SwiftFormat
-   SwiftGen
-   Periphery

-   iOS 16.0
